Tutorials

How to Post Content in Groups.

Text

  1. In the Text tab, click the Post Something button.
  2. Then type your text in the textbox and click post button.

Picture

  1. In the Picture tab, click the Post Picture button.
  2. Click select file or drag the file you wish to post in the drop files here box.
  3. Then type in your caption in the caption textbox then click post button.

Video

  1. In the Video tab, click the Post Video button.
  2. Click select file or drag the file you wish to post in the drop files here box.
  3. Then type in your caption in the caption textbox then click post button.

How to like a post.

  1. Navigate to the post you wish to like.
  2. When you hover over the post, like button will appear.
  3. Click like button

How to Unlike a post.

  1. Once the post is liked the like button will change to unlike.
  2. To unlike the post you click on the unlike button.

How to comment on a post.

  1. Navigate to the post you wish to comment on.
  2. When you hover over the comment, comment button will appear and click comment.
  3. Then type your text in the textbox and click comment button

How to delete a comment.

  1. Navigate to the comment you wish to delete.
  2. When you hover over the post, on the top corner an X will appear.
  3. click it and comment will be deleted.

How to create an event.

  1. In the events tab, click the create event button.
  2. The create event window pop up will be displayed.
  3. Enter all the required information and click done button
